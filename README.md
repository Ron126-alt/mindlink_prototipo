#MindLink - Monitoreo de Salud Mental

##Descripción
MindLink es una aplicación web orientada al monitoreo de la salud mental de los usuarios. Permite registrar emociones, visualizar información en un dashboard y generar recomendaciones de apoyo emocional.

##Objetivo
Brindar a los usuarios una herramienta que les permita hacer seguimiento a su estado emocional y recibir contenido que los ayude a mejorar su bienestar.

##Tecnologías utilizadas
- JavaScript
- Node.js
- MySQL
- HTML, CSS

##Funcionalidades
- Registro de usuarios
- Inicio de sesión
- Registro de emociones
- Dashboard
- Recomendaciones
- Notificaciones

##Estructura del proyecto
- `/backend` → lógica del servidor
- `/js` → scripts del frontend
- `/css` → estilos
- Archivos `.html` → vistas

##Documentación
La documentación del proyecto se encuentra en la carpeta `/docs`.
